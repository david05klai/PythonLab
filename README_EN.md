### Description
**PythonLab** is a learning-focused Python project that explores basic programming logic through small interactive games and exercises.  
It includes projects such as **Number Guessing** and **Word Guessing**, demonstrating the use of functions, conditionals, loops, and user input.

The main goal of this project is to practice Python fundamentals and understand how to structure small applications into multiple modules.

---

### Features
- Number guessing game  
- Word guessing game  
- Modular code structure (menu, logic, and input management)  
- Simple and clear console interface  

---

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/david05klai/PythonLab.git
   ```
2. Enter the project directory:
   ```bash
   cd PythonLab
   ```
3. Run the main program:
   ```bash
   python Main.py
   ```
